Répertoire
```cmd
cd JeuDesOrbites
```

Compilation
```cmd
javac -d .\build\classes\jeudesorbites\ .\src\jeudesorbites\*.java
```

Execution
```cmd
java -cp .\build\classes\ jeudesorbites.JeuDesOrbites
```

## Pour les caractères spéciaux sous Windows

Activer les caractères ASCII dans le terminal : 
```cmd 
chcp 65001
```
